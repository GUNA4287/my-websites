🔹 Overview – Hidden Web Application Hidden is a unique and interactive web application developed using HTML, CSS, JavaScript, and Bootstrap, built within Visual Studio Code. While it follows the structure of an **e-commerce platform, Hidden is not a traditional shopping site. Instead, it introduces an innovative reward-based virtual coin system, combining entertainment with product purchasing in a gamified way. This project is currently 67% completed and is still under active development as part of your ongoing work. The system is designed to reward users with virtual coins for viewing advertisements, which can then be used to "purchase" listed products. It offers a fresh take on user engagement and online rewards, simulating a digital economy centered around activity and interaction rather than real currency alone. 🌐 Technology Stack: * Frontend : HTML, CSS, JavaScript, Bootstrap * Development Environment : Visual Studio Code * Application Type : Web-Based (Client-Side) ⭐ Key Features: 🎥 Watch Ads to Earn Coins Users can watch short video advertisements to earn virtual coins. Each product on the platform includes a "Watch Video" button. Upon viewing the ad and passing a simple CAPTCHA verification, users are rewarded with random types of coins. 💰 Virtual Coin System There are five types of virtual coins used throughout the application: * TOYO * TAKKEY * GDEO * MAXI * Fifth type based on design Each product requires a specific amount and type of coin to purchase. For example: A product like an LED TV priced at ₹5000 may also show a 2000 TOYO coin equivalent at the bottom of its card. This gives users two options: 1. Pay with traditional online payment methods (e.g., UPI, PayPal, debit/credit). 2. Redeem and purchase using the collected virtual coins. 🛒 Product Catalog Products on the platform simulate a store model, but are intended for demonstration of coin-based engagement. Each item: * Displays the price in currency and coin * Includes a "Watch Video" option for earning coins * May offer multiple purchase paths (money or coin-based) 🧑‍💼 Profile and Hidden Card Feature Users have a personal profile page that displays: * User details (name, email, etc.) * Coin balances * Access to the unique “Hidden Card" feature The Hidden Card: * Appears as a virtual card on the profile page * Requires a secure PIN to unlock * Once unlocked, the card flips to reveal detailed balances of each coin type This enhances the sense of privacy and exclusivity in the coin collection process, adding an element of discovery and security. 🛡️ User Experience & Interface * 🔓 Simple Navigation: Intuitive layout with Bootstrap components ensures smooth navigation across the platform. * 🎨 Modern UI Design: Clean, responsive, and mobile-friendly interface using CSS styling and Bootstrap themes. * 🔐 Security Measures: CAPTCHA validation before coin rewards and PIN access for the Hidden Card reinforce system integrity. 🎯 Project Goals & Benefits * 💡 Gamified Commerce Experience: Encourages user interaction through a reward-for-attention model. * 🎁 Engaging Coin Economy: Mimics a digital currency environment while remaining educational and fun. * 📈 Monetization Potential: Concept can be expanded into real ad-based revenue models or product partnerships. * 🧪 Creative Experimentation: Blends e-commerce, advertising, and gaming elements into one creative application. 🛠️ Development Status * Current Completion : \~67% * Stage : Ongoing Development * Next Phases : Backend integration (if applicable), coin value logic refinement, user data storage, and final UI polishing This project demonstrates your creativity, frontend development skills, and ability to design interactive user experiences. The blend of virtual currency, ad-driven rewards, and a polished interface makes "Hidden" a unique and engaging application with real-world potential in advertising and entertainment platforms.
